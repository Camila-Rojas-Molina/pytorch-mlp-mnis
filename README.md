# PyTorch MLP — MNIST Digit Classification

Intro to PyTorch covering tensors, autograd, and training a 
configurable Multilayer Perceptron (MLP) on the MNIST dataset.

## Topics covered
- PyTorch tensors and math operations
- Automatic differentiation with `backward()`
- GPU vs CPU with CUDA
- Custom `nn.Module` with variable hidden layers
- Training loop with DataLoader, CrossEntropy, Adam

## Run it
Open in Google Colab or locally with Jupyter.

pip install -r requirements.txt

GPU recommended for training, but CPU works fine for this dataset size.
